## Iniciando o projeto

Para inciar, clone o repo e insira os seguintes comandos:

```bash
npm install
npm run dev
# or
yarn install
yarn dev

```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador e veja o resultado.

escolhidas por você para concluir o desafio.
